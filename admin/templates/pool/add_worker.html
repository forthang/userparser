How to get Session String

To create a worker, you need a Pyrogram session string from a Telegram account.

Steps:
  1. Install Pyrogram: pip install pyrogram
  2. Run this Python script: 
    from pyrogram import Client

api_id = YOUR_API_ID
api_hash = "YOUR_API_HASH"

with Client("session", api_id, api_hash) as app:
    print(app.export_session_string())
  3. Login with the phone number
  4. Copy the session string
Important: Use separate accounts for workers. Don't use your personal account.
